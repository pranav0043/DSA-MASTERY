## 🗂️ Quick Reference

### Arrays
- Techniques: Two Pointer, Sliding Window, Prefix Sum
- 📝 *Most problems start with arrays — practice daily*

### Linked List
- Singly, Doubly, Circular
- Must know: Reverse, Detect Cycle (Floyd's Algorithm)

### Stack
- LIFO | Uses: Expression eval, DFS, Undo
- 📝 *Practice: Infix → Postfix conversion*

### Queue
- FIFO | Circular Queue, Deque, Priority Queue
- Uses: BFS, Scheduling

### Trees
- Binary Tree, BST, AVL
- Traversals: Inorder, Preorder, Postorder, Level Order
- 📝 *Revise: Height, Diameter, LCA*

### Heap
- Min/Max Heap, heapify()
- Used in: HeapSort, Top-K problems

### Hashing
- Collision: Chaining vs Open Addressing
- Java: `HashMap`, `HashSet`, `LinkedHashMap`

### Graphs
- BFS, DFS, Dijkstra, Bellman-Ford, Kruskal, Prim
- 📝 *Know: when to use BFS vs DFS*

### Sorting — Complexity Table

| Algorithm      | Best       | Worst      | Stable |
|----------------|------------|------------|--------|
| Bubble Sort    | O(n)       | O(n²)      | ✅     |
| Selection Sort | O(n²)      | O(n²)      | ❌     |
| Insertion Sort | O(n)       | O(n²)      | ✅     |
| Merge Sort     | O(n log n) | O(n log n) | ✅     |
| Quick Sort     | O(n log n) | O(n²)      | ❌     |
| Heap Sort      | O(n log n) | O(n log n) | ❌     |

### Searching
- Linear: O(n) | Binary: O(log n) — sorted array only
- 📝 *Binary search on answer — important advanced trick*

### Recursion & Backtracking
- N-Queens, Sudoku, Rat in Maze
- 📝 *Always draw recursion tree while solving*

### Dynamic Programming
- Memoization vs Tabulation
- Key problems: Knapsack, LCS, LIS, MCM, Coin Change
- 📝 *Check: overlapping subproblems + optimal substructure*

### Greedy
- Activity Selection, Fractional Knapsack, Huffman Coding
- 📝 *Justify why greedy works — don't just assume*

### Trie
- Insert/Search: O(L) where L = word length
- Uses: Autocomplete, Spell check
